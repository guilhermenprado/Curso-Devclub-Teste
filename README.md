**Project**

- **Name**: Curso Devclub Teste
- **Description**: A small front-end test project containing a static web page with related styles and JavaScript. It demonstrates a simple UI and client-side behavior using vanilla HTML, CSS, and JavaScript.

**Technologies**

- **HTML**: Structure and markup (`index.html`).
- **CSS**: Styling (`styles.css`).
- **JavaScript**: Client-side logic (`script.js`).

**Project Structure**

- `index.html` — main HTML file
- `styles.css` — stylesheet for layout and design
- `script.js` — JavaScript for interactivity

**Usage**

- Open `index.html` directly in your web browser for a quick preview.
- Or run a simple local server for proper relative requests:

  - Using Python 3:

    ```bash
    python -m http.server 8000
    ```

  - Then open `http://localhost:8000` in your browser.

**Development**

- Edit the files in your preferred editor and refresh the browser to see changes.
- For a live-reload experience, use the VS Code Live Server extension or similar tooling.
